  - currentSize라는 state를 객체로 만들어서 사용하고 있었는데, isFull이라는 값이 불필요하게 계속해서 쓰이고 있던 것을 없앴다. 하지만 객체에 값을 넣지 않으면 eslint에서 잘못된 것이라고 인식하므로 인터페이스를 추가해주었다.

    ```jsx
    export interface ISize {
      w: number;
      h: number;
      isFull?: boolean;
    }
    ```
